## Шахматы
Увлекательная игра шахматы с интересным дизайном и проработанной логикой
## Сборка
1) убедитесь, что у вас установлен компилятор C++, Qt5 и qmake
2) введите команду и ожидайте сборки
```bash
bash Build
```
## Запуск
```bash
bash Run
```
